Question-1: What are the different ways to select an element in the DOM?

There are five ways, we can select an element or elements in the Dom. (i)getElementsByTag (ii) getElementById, (iii) getElementsByClass, (iV) querySelector & (V) querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: All the text that we can find for any elements including all the gaps and innerHTML tags.
innerText: Only the text content inside of any elements.
textContent: In JavaScript, textContent is a property of the Document Object Model (DOM) that allows you to get or set the text content of a specified node and all its descendants.

Question-3: What is event delegation in the DOM?

Event delegation in JavaScript is a technique where a single parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is when an event starts on a child element and automatically propagates up to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: We use createElement to create necessary elements.
Add: We use appendChild() to add created elements into our desired location in the code.
Remove: We use element.remove() or parent.removeChild(element) to delete anything from any location.
